查询24小时收视

基本信息

Path： /screenservice/query

Method： POST

接口描述：

功能：按小时查询全国24小时用户收视情况，收视人数，收视次数和收视时长，按照日期，时段倒序排序。时间范围为指定时间范围，默认最近7天
描述：用户24小时收视取自bto_c3.BKR_USER_USERVIEWH24，选出最近七天，并根据hourid分组聚合
接口类型：Oracle

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
qdi text 是 queryHourUserView qdi接口名称
userid text 是 superman 访问用户id
token text 是 HDdt7Zhq5qjWzddDMAni9u1G 访问用户token
limit text 是 20 分页参数每页大小
offset text 是 0 分页参数起始位置
starttime text 是 20200101 开始日期，格式YYYYMMDD
endtime text 是 20200107 结束日期，格式YYYYMMDD

返回数据

名称类型是否必须默认值备注其他信息
codenumber必须错误代码
exportstring非必须
msgstring非必须错误描述
subjectobject []必须返回数组

item 类型: object

├─ MONTHDAYstring必须日期，格式YYYYMMDD
├─ HOURIDstring非必须24小时时段，值为0~23
├─ USERCOUNTstring非必须收视人数
├─ VIEWCOUNTstring非必须收视次数
├─ VIEWTIMEnumber非必须收视时长
totalstring必须总记录数

查询开机情况

基本信息

Path： /screenservice/query

Method： POST

接口描述：

功能：查询区域用户开机情况，时间范围为最近一天
描述：首先开机率=区域内开机用户数/区域内总用户数，其中开机用户数取自bto_c3.bkr_user_useractivity的bootuser，区域用户数取自BKR_R002_USERAREATREND的user012，只需要两表按照areacode分组后sum指定的字段，并关联上区域信息表bkrvw_cfg_cucc_areacode即可得到areaname。
数据来源：用户活跃 bto_c3.bkr_user_useractivity，用户发展BKR_R002_USERAREATREND
接口类型：Oracle

查询收视排行榜

基本信息

Path： /screenservice/query

Method： POST

接口描述：

功能：查询直播，点播，回看收视排行榜，时间范围为最近一天
描述：根据viewtype不同查询不同的数据源，收视类型，1：直播，2：点播，3：回看。
其中直播节目取自bi_viewlog_c3.BKR_CHANNEL_SCHEDULE_LIANTONG_HISTORY,点播取自bi_viewlog_c3.bkr_vod_view_history，回看取自bi_viewlog_c3.bkr_tvodtvbyvod_history
其中点播节目名取自bi_viewlog_c3.view_pl_vsp_media_hadoop_history，回看节目名来自bi_viewlog_c3.bkr_schedule
接口类型：Presto

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
qdi text 是 queryViewTopN qdi接口名称
userid text 是 superman 访问用户id
token text 是 HDdt7Zhq5qjWzddDMAni9u1G 访问用户token
limit text 是 20 分页参数每页大小
offset text 是 0 分页参数起始位置
viewtype text 是 1 收视类型，1：直播，2：点播，3：回看

返回数据

名称类型是否必须默认值备注其他信息
codenumber必须错误代码
exportstring非必须
msgstring非必须错误描述
subjectobject []必须返回数组

item 类型: object

├─ PROGRAMNAMEstring必须节目名称
├─ CHANNELNAMEstring非必须频道名称，viewtype=1或3时有效
├─ RTGstring非必须收视率，viewtype=1有效
├─ RATIOstring非必须收视占比viewtype=2或3有效
├─ viewusernumber非必须收视人数
totalstring必须总记录数
